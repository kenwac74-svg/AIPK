# AIPK Public Static Mockup

This folder contains only the public static mockup build extracted from `AIPK-main.zip`.

Included:
- `index.html`
- `wallet-preview.html`
- `.nojekyll`
- `assets/*.js`
- `assets/*.css`

Excluded for security:
- Packaged source ZIP files
- Admin source package
- Backend/server source
- GitHub Actions workflows
- Environment files
- Database backups
- Credentials, keys, tokens, and deployment scripts

Use this folder as the root of the replacement public GitHub Pages repository.
